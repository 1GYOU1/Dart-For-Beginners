## Dart 초기 세팅

- flutter를 설치하면 dart는 자동으로 같이 설치됨.

### ✔ Mac

- (brew 설치: https://brew.sh/index_ko)
<br> 
>$ brew install flutter

### ✔ window
- (chocolatey 설치: https://community.chocolatey.org/)
<br>
> $ choco install flutter

<br>

### vscode extensions 설치
- Flutter 검색, 설치

<br>
